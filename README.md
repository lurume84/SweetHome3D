## About

**Sweet Home 3D** is a free interior design application that helps you place your furniture on a 2D house plan, with a 3D preview.

This repository is an **unofficial Git mirror** of the official SVN repository hosted on SourceForge, converted using `git-svn`.

| | |
|---|---|
| **Original SVN** | https://svn.code.sf.net/p/sweethome3d/code/ |
| **Mirrored up to** | SVN revision **r9047** |
| **Latest version** | **7.5.2** (`tag: V_7_5_2`) |

| **Official site** | https://www.sweethome3d.com |
